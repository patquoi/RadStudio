INFORMATIONS IMPORTANTES - DUPLICATA
____________________________________


LICENCE

Duplicata est soumis à la Licence Publique Générale version 2.
Veuillez vous rapporter aux fichiers license.txt et licence.txt.

CONFIGURATION REQUISE

L'application nécessite environ 32 Mo de mémoire RAM en version 32 bits
                     et environ 64 Mo de mémoire RAM en version 64 bits.
L'installation nécessite environ 20 Mo d'espace disque.
La résolution graphique minimale doit être de 800x600x256 couleurs.
Cependant, une résolution de 1024x768 est fortement recommandée.

CONTENU

Duplicata est fourni :
 - avec l'ODS 9ème édition (2024) comme dictionnaire de référence,
 - avec l'archive compressée des sources du programme (Duplicata.zip).

DESINSTALLATION

Vous avez deux manières de désinstaller Duplicata :
 - Soit en allant dans le panneau de configuration de Windows puis en 
   choisissant Ajout/Suppression de programmes et pour finir, en double 
   cliquant sur Duplicata
 - Soit en allant dans le menu Démarrer | Programmes | Patquoi.fr | Duplicata | 
   Désinstallation (chemin par défaut)
Tous les fichiers générés entre l'installation et la désinstallation 
NE SONT PAS SUPPRIMES (fichier de configuration et de paramètres, parties
sauvegardées, fichiers HTML...)

IMPORTATION DE LA CONFIGURATION D'UNE VERSION ANTERIEURE

À la fin de l'installation, vous pourrez importer la configuration d'une
version antérieure à la version 1.8.4 (version 32 bits). Assurez-vous que 
la case "Importer une configuration déjà existante" est cochée.
  
AIDE

Une aide HTML est disponible. Vous avez plusieurs manières d'y accéder :
 - Si Duplicata est lancé, appuyez sur F1 ou choisissez la rubrique ? | Aide du
   menu principal ;
 - Choisissez la rubrique Aide du menu Duplicata dans le menu Démarrer de 
   Windows (le chemin par défaut est Programmes | Patquoi.fr | Duplicata),
 - Ouvrez le fichier index.html dans le dossier Patquoi.fr\Duplicata\html 
   (chemin par défaut) du dossier Windows "Program Files". Autrement dit, le 
   fichier index.html est situé dans le dossier html depuis l'endroit où se 
   situe le programme Duplicata.exe.

SOURCES

Les sources du programme sont dans l'archive Duplicata.zip.
La compilation nécessite RadStudio 10 minimum (version Community gratuite).

VERSIONS DES PARTIES SAUVEGARDEES

 - Il n'est pas possible d'ouvrir une partie enregistrée avec une version de
   Duplicata supérieure ou égale à la version 1.5 (respectivement 1.5.3), 
   avec une version antérieure à la version 1.5 (respectivement 1.5.3).
 - Les parties enregistrées avec une version antérieure à la version 1.5 ne
   peuvent pas avoir d'évaluation du niveau de difficulté lorsqu'elles sont
   reprises avec une version supérieure ou égale à la version 1.5 mais peuvent
   néanmoins être poursuivies avec une version 1.5 et supérieure.
 - Les parties enregistrées avec une version antérieure à la version 1.4 ne 
   peuvent plus être chargées par la version 1.5.1 ou ultérieure.
 - Les parties enregistrées avec une version comprise entre la version 1.5 et
   1.5.2 (incluses), qui s(er)ont enregistrées avec une version 1.5.3 ou 
   ultérieure s(er)ont du format de la nouvelle version.

FICHIERS HTML GENERES 

Les fichiers HTML "Feuille de match détaillée", "Liste des parties jouées" et
"Liste des records" utilisent des fichiers images de type bitmap situés dans 
le sous-dossier "bin/html" du dossier d'installation de Duplicata. Si vous 
désinstallez Duplicata et que vous souhaitez conserver les fichiers HTML, ces
images ne seront pas accessibles. Pour avoir les images dans les pages HTML, 
procédez comme suit :
 - Si Duplicata a été désinstallé, réinstallez-le ;
 - Copiez les onze fichiers d'extension bmp (image bitmap) dans le même 
   dossier que celui des fichiers à conserver ;
 - Dans les fichiers HTML, recherchez et supprimez toutes les occurrences du
   type "file:///[...]/Duplicata/html/" (sans les guillemets). Par défaut,
   il s'agit de "file:///C:/Program%20Files/Patquoi.fr/Duplicata/html/".
