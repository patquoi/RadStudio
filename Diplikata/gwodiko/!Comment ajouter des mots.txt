COMMENT AJOUTER DES MOTS DANS GWODIKO
-------------------------------------

 1. Ouvrir le fichier Gwodikovilsen.txt
 2. Rajouter les mots dans l'ordre alphabétique (attention aux accents, exemple brèn pour distinguer les sons on/en/in)
 3. Lancer KreGwodiko depuis Delphi et coller le contenu du fichier ci-dessus dans le mémo et appuyer sur "Analyser"
 4. Si OK, appuyer sur "Enrichir" puis copier-coller le résultat dans le fichier "Gwodikoenrichi.txt".
 5. Appuyer sur "Générer" puis copier-coller les infos depuis la ligne "Création du fichier final..." dans le fichier enfogwodikovilsen.txt
 6. Corriger les différences dans les deux lignes de code de déclaration (NbLettresDico et nbl)
 6b.Appuyer sur Générer JS
 7. Appuyer sur "Calculer stats" et mettre à jour le fichier statsfrq.txt en remplaçant les lettres Â/Ç/Ê/Î/Ñ/Ô/Û par An/Ch/En/Ui/Ng/On/Ou 
 8. Double cliquer sur copy2rechdiko.bat
 9. Modifier le code de Diplikata et Rechdiko avec les deux nouvelles déclarations modifiées ci-dessus.
10. Recompiler et exécuter Rechdiko pour générer le dictionnaire de recherche. 
11. Double cliquer ensuite sur !concatene&copie.bat (dans rechdiko/bin) 
12. Recompiler Diplikata et tester les nouveaux mots dans la recherche de mots.
13. Générer l'Ofisyeltiraj à partir des fichiers Z*.txt (convertir en UTF8) 
    qui se trouvent dans rechdiko/bin (vider/coller sur chaque feuille ods)
14. Faire le dictionnaire Kreyòl<->Eskrab à partir du fichier Ofisyeldiksyone.txt :
     - convertir en UTF-8 avec notepad++
	 - Ouvrir Ofisyeldiksyone.ods sous OpenOffice (Mac) 
     - effacer puis coller le fichier Ofisyeldiksyone.txt 
     - retailler automatiquement les colonnes
	 - changer l'en-tête (nombre d'entrées = lignes des fichiers gwodikovilsen{enrichi}.txt)
	 - exporter en PDF
14b.Copier les fichiers ods et pdf générés respectivement dans gwodiko/bin et rechdiko/bin 
15. Convertir les fichiers rechdiko\bin\Anagrammes.js+Tirages.js et gwodiko\dico.js et diko.js en UTF8 pour les apps mobiles
16. Double-cliquer sur Gwodiko\CopyJS.bat => Dossier partagé VMWare
17. Générer Insdiplikata.exe en double-cilquant sur Nettoie.bat puis Prépare.bat sans oublier d'incrémenter la version (MyAppVersion) correspondant aux exécutables
18. Copier sur MaPage les pdf et insdiplikata et modifier index.html
19. Sauvegarder Diplikata xxxx (Rio).zip et générer www.zip pour copie sur Lacie
20. Mettre à jour gitHub.com
