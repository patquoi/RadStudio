COMPATIBILITE WINDOWS

Vous pouvez installer Le Compte Pour De Bon de Windows XP à Windows 10.
 
INSTALLATION ET DESINSTALLATION

L'archive d'installation a été générée avec l'outil GPL InnoSetup.
Il permet d'installer "Le Compte Pour De Bon" avec les sources.
Elle inclut la désinstallation. Pour ce faire, aller dans le panneau de 
configuration et choisir Ajout/Suppression de programmes puis 
Le Compte Pour De Bon. Sous Windows 10, aller dans les paramètres puis 
Programmes et rechercher "Le Compte Pour De Bon".

Les paramètres de jeu NE SONT PAS SUPPRIMES lorsque vous désinstallez le 
programme. Après désinstallation, vous devez aller supprimer explicitement le
dossier où a été installé le programme.

CONFIGURATION MINIMALE ET RECOMMANDEE

Pour évaluer le compte le plus gourmand en retirant toutes les optimisations
paramétrables, donnant 4 162 solutions, il faut 130 méga-octets de mémoire.
Il est donc recommandé d'avoir une mémoire d'au moins 512 méga-octets pour ne
pas avoir de manque de mémoire dans le cas décrit précédemment sous Windows XP
et sans autre (gros) programme chargé en mémoire. 

La résolution de l'écran minimale est de 1024x768.

PERSONNALISATION DES STATISTIQUES

Si plusieurs joueurs utilisent la même machine sur lequel doit être installé
Le Compte Pour De Bon, il est conseillé de l'installer autant de fois qu'il
y a d'utilisateurs. En effet, si plusieurs joueurs utilise le même jeu, 
les statistiques seront communes. Pour obtenir des statistiques par joueur,
installez autant de fois le programme que de joueurs différents sur la même
machine. Ainsi, lors de l'installation, il faut modifier deux paramètres :
 - Lorsque le chemin du DOSSIER D'INSTALLATION s'affiche, ajoutez le prénom 
   par exemple, à la fin du chemin proposé :
     C:\Program Files\Ihsv\Le Compte Pour De Bon\<Prénom>
 - Pareil lorsque le DOSSIER DU MENU DÉMARRER s'affiche (panneau suivant):
     Ihsv\Le Compte Pour De Bon\<Prénom>
IMPORTANT : Au moment de désinstaller le programme dans le cas où plusieurs
instances du programme sont installées, il faut utiliser le raccourci  
"Désinstallation" situé dans le menu "Démarrer\Programmes" et non pas celui 
du panneau de configuration "Ajout/Suppression de programmes" car celui-ci 
désinstallerait la dernière instance uniquement.

SOURCES

La version 32 bits du Compte Pour De Bon a été développé avec Delphi 2005. 
À partir de la version 1.3 (32/64 bits), il a été développé avec RAD Studio
10.3.3 Rio Community Edition (Delphi) qui est une version gratuite.

Pour les installer, il faut décompresser le fichier src.zip dans le dossier
d'installation en précisant de recréer les sous-dossiers. Les sources sont
alors décompressés dans le dossier src.

Les sources peuvent donc être recompilés avec les versions plus complètes de
Delphi. Pour les autres versions de Delphi, cela ne devrait pas poser 
trop de problèmes vu qu'il n'y a pas de composants spéciaux utilisés. 
Il se peut qu'il y ait quelques propriétés supplémentaires dans les fichiers
ressources des fiches (*.dfm) mais pas inévitables. Aucun projet avec des 
versions antérieures à RAD Studio 10.3.3 n'a été créé ni testé à ce jour.
